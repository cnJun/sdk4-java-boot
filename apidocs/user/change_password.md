# 修改密码

修改用户登录密码

## 公共请求参数

|参数          |值
|-------------|-------
|method       |user.change_password

## 业务请求参数

|参数                 |是否必填 |描述
|--------------------|--------|----
|old_password        |是      |旧密码
|new_password        |是      |新密码(6-12位)

## 业务响应参数

|参数              |是否必填 |描述
|-----------------|--------|----
|id               |是      |用户ID
|mobile           |是      |注册手机号码
