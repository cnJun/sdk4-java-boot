# 二维码图片

二维码图片生成接口，该接口以 `GET` 方式请求，返回 url 参数指定的数据和大小的二维码图片，该 url 链接可直接作为 `img` 的 `src` 使用。

接口地址：`/c/qrimg`

## 请求参数

|参数           |类型      |必须  |说明                |
|--------------|---------|------|-------------------|
|data          |String   |是    |二维码图片内容，需要进行URL编码
|w             |Integer  |否    |二维码图片宽度
|h             |Integer  |否    |二维码图片高度

*未指定图片的宽度和高度时，后端会根据 data 内容自动计算*

示例：http://xxx.com/c/qrimg?data=hello&w=500&h=500
